Introduction

This guide is designed to help you navigate the resources in this API and show you how to make different types of queries. Please note that while the GET methods can be accessed directly from this page, the POST, PUT, and DELETE methods require the use of Swagger.

Post an artist

You can post an artist by adding all parameters: artistName, genre, image, description, and albums. It is possible to post without an image, but note that it's not recommended. Id is auto-incremented by default, and can be left at 0.

Example

POST: Loading... 

          
{
  id: 0,
  artistName: "string",
  genre: "string",
  image: "string",
  description: "string",
  albums: [
      {
          id: 0,
          albumName: "string",
          artistId: 0,
          albumImage: "string",
          songs: [
              {
                  id: 0,
                  songName: "string",
                  albumId: 0
              },
              ...
          ]
      },
      ...
  ]
}

Update an artist

You can update an artist by altering the parameters you want to change. You won't be able to alter the id as it is auto-incremented in the database, but artist name, genre, image, description, and albums can be changed.

Example

PUT: Loading... 

          
{
  id: 1,
  artistName: "string",
  genre: "string",
  image: "string",
  description: "string",
  albums: [
      {
          id: 1,
          albumName: "string",
          artistId: 1,
          albumImage: "string",
          songs: [
              {
                  id: 1,
                  songName: "string",
                  albumId: 1
              },
              ...
          ]
      },
      ...
  ]
}

Delete an artist

You can delete an artist by providing the id of the artist you want to delete from the database.
